# 上海高中成绩单教学分析 PDF 报告生成 Skill

## 概述

`shanghai-high-school-score-report` 是一个用于处理上海高中常见成绩单 Excel、自动生成正式教学质量分析 PDF 报告的 CLI 工具。

## 使用场景

1. 两个或多个班级整体成绩对比分析
2. 只分析女生群体，生成女生专项 PDF
3. 只分析男生群体，生成男生专项 PDF
4. 分析某一个班内部男女生表现差异
5. 生成正式教学质量分析报告 PDF
6. 根据用户补充的性别规则生成性别字段

## 输入要求

### Excel 文件
- 格式：`.xlsx` / `.xlsm` / `.xls`
- 第一行为列名（表头）

### 用户需提供
- 班级文件路径
- 班级名称
- 性别规则（可选）
- 报告类型
- 输出文件名（可选）

### 性别规则格式

```
--gender-rules "7班:male_first=23;8班:male_first=22"
```

含义：7班前 23 条记录为男生，其余为女生；8班前 22 条记录为男生，其余为女生。

### 报告类型

| 类型 | 说明 |
|------|------|
| `overall` | 多班级整体对比报告 |
| `female_only` | 只分析女生群体 |
| `male_only` | 只分析男生群体 |
| `class_gender_compare` | 分析指定班级内部男女生差异 |
| `custom_group` | 预留接口 |

## 上海高中成绩单字段规则

### 基础信息
- 系统号、班级、座号、姓名、性别（如果存在）

### 单科成绩
- 语文成绩、数学成绩、外语成绩
- 物理成绩、化学成绩、政治成绩、历史成绩、地理成绩、生物成绩、信息科技成绩

### 单科排名
- 语文班次、语文年次、数学班次、数学年次
- 其他科目班次 / 年次

### 汇总成绩与排名
- 总分3、总分10
- 总分3班次、总分3年次、总分10班次、总分10年次

## 总分3 / 总分10 的关键处理规则

**这是本 Skill 最重要的规则：**

1. 总分3 ≈ 语文 + 数学 + 外语三科总分，满分约 300
2. 总分10 ≈ 十门科目综合总分，满分约 1000
3. **总分3、总分10 不能直接和 90 / 80 / 60 比较**
4. 必须自动生成：
   - **三科平均 = 总分3 / 3**
   - **十科平均 = 总分10 / 10**
5. 优秀率、优良率、及格率只能基于以下百分制字段计算：
   - 单科成绩
   - 三科平均
   - 十科平均

### 错误做法（严禁）
- `总分3 >= 90`
- `总分10 >= 80`
- `总分3 >= 80`

### 正确做法
- 单科成绩使用 90 / 80 / 60 判断
- 三科综合使用"三科平均"判断
- 十科综合使用"十科平均"判断

### 排名处理
- 总分3班次、总分3年次、总分10班次、总分10年次属于排名字段
- 只能进入排名分析，不得进入平均分、优良率、分布图等成绩图表

### 报告说明
PDF 报告中会说明：
> "为保证不同分值尺度下的可比性，本报告将总分3转换为三科平均，将总分10转换为十科平均后，再进行优秀率、优良率、及格率及分布分析。"

## 默认评价标准

| 等级 | 标准 |
|------|------|
| 优秀 | ≥ 90 |
| 优良 | ≥ 80 |
| 及格 | ≥ 60 |
| 低分 | < 60 |

### 临界生定义

| 类型 | 分数区间 |
|------|----------|
| 冲优秀临界 | 85–89 |
| 冲优良临界 | 75–79 |
| 及格临界 | 55–59 |
| 风险学生 | < 55 |

可通过 `--thresholds` 参数修改评价标准。

## 分析指标体系

1. **数据概况** — 班级数量、分析对象、学生人数、科目数量
2. **整体成绩表现** — 三科平均、十科平均、平均分、中位数、极差、标准差
3. **达标情况** — 优秀率、优良率、及格率、低分率、分数段结构
4. **主科基础** — 语文、数学、外语平均分、三科平均、三科优良率
5. **十科综合** — 十科平均、十科优良率、综合水平分层
6. **单科结构** — 各科平均分、优良率、及格率、优势/薄弱/分化学科
7. **年级竞争力**（有年次字段时）— 年级百分位、前10%/25%、后25%
8. **学生分层** — 优秀层、优良层、中间层、临界层、待提升层、风险学生
9. **偏科与均衡性** — 偏科指数、主科稳定指数
10. **性别分析**（提供性别规则时）— 男女生横向对比

## PDF 报告结构

1. 封面（标题、分析对象、生成时间、数据来源）
2. 摘要（分析对象、主要发现、核心建议）
3. 一、数据说明与分析口径
4. 二、整体表现分析
5. 三、达标情况分析
6. 四、主科基础分析
7. 五、十科综合与单科结构分析
8. 六、排名与年级竞争力分析（如果有排名字段）
9. 七、分层与临界生分析
10. 八、偏科与均衡性分析
11. 九、性别专项分析（如果启用且未过滤性别）
12. 十、主要发现与结论
13. 十一、教学建议
14. 结语

## 报告风格

- 教学质量分析报告
- 类似小型论文式汇报
- **文字为主，图表为辅**
- 每张图服务于正文分析
- 图表插入对应章节
- 有明确的数据依据和分析判断

## 调用示例

```bash
# 多班级整体对比
python generate_score_report.py \
  --files "7班整体成绩.xlsx" "8班整体成绩.xlsx" \
  --class-names "7班" "8班" \
  --output "整体分析报告.pdf"

# 女生专项分析
python generate_score_report.py \
  --files "7班整体成绩.xlsx" "8班整体成绩.xlsx" \
  --class-names "7班" "8班" \
  --gender-rules "7班:male_first=23;8班:male_first=22" \
  --report-type "female_only" \
  --output "女生专项分析报告.pdf"

# 男生专项分析
python generate_score_report.py \
  --files "7班整体成绩.xlsx" "8班整体成绩.xlsx" \
  --class-names "7班" "8班" \
  --gender-rules "7班:male_first=23;8班:male_first=22" \
  --report-type "male_only" \
  --output "男生专项分析报告.pdf"

# 7班男女生对比
python generate_score_report.py \
  --files "7班整体成绩.xlsx" \
  --class-names "7班" \
  --gender-rules "7班:male_first=23" \
  --report-type "class_gender_compare" \
  --target-class "7班" \
  --output "7班男女生对比报告.pdf"
```

## 降级处理

如果 Playwright 不可用（无法自动生成 PDF）：
1. 工具会保留完整的 HTML 报告（`print_report.html`）
2. 用户用浏览器打开 HTML 文件
3. 按 Ctrl+P 打印为 PDF
4. 设置：A4、背景图形开启、边距最小
