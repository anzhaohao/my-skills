# 硬件与服务器信息 Skill

这个 skill 用来记录 An Zhaofeng 的长期硬件资产信息，包括服务器、电脑、GPU、实验设备、采集卡、相机、存储设备和网络设备。

## 什么时候使用

- 用户提供新的服务器、电脑、实验设备或硬件参数时。
- 用户询问某台服务器、电脑或设备是否适合运行某个任务时。
- 需要部署服务、连接设备、排查硬件环境时。
- 需要补全或核对已有硬件信息时。

## 维护规则

- 新硬件信息出现时，自动补充到这个 skill。
- 不确定的信息标记为“待补充”，不要猜。
- 严禁记录密码、API key、token、私钥、cookie、恢复码等私密信息。
- 服务器 Docker 部署位置和运维习惯不写在这里，写到 `azf-server-deploy`。
- 每次更新 `SKILL.md` 后，同步更新这个 README。
