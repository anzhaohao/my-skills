# 固定概念类型

只允许：基础概念、物理量、物理机制、实验现象、测量方法、数值算法、数据表示、实验器件、工程问题。不得由 Agent 临时发明新的类型名称。

# 固定状态

- `待整理`：已有内容，但结构、来源或表述尚未完成。
- `已整理`：可以正常阅读和跨论文复用。
- `待合并`：发现重复卡，需要保留独有内容后合并。
- `待核对`：解释、来源或归类存在不确定性。

# 新建和更新规则

1. 新建前搜索文件名、英文名和 `aliases`。
2. 已有概念必须更新中央卡，不再创建论文内部副本。
3. 一张卡可以有多个领域和主题，分类变化只改属性，不移动文件。
4. 论文中的具体语境追加到“在不同论文中的用法”。
5. 疑似重复时标记为 `待合并` 或 `待核对`，不得直接覆盖。
6. 合并前检查所有来源卡、反向链接和独有内容。
7. Templater 模板只保存在 `05-Junk Drawer/2_模板/2.1 Templater/概念卡模板.md`。
