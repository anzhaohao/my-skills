# 用途

这个 Skill 用于把论文段落制作成逐句精读笔记，并用 PDF++ selection 链接保留句级来源。

# 两轮位置规则

第一轮调用主文献工作流的 `locate` 命令，只展示论文根目录和中央扫盲班位置，不创建笔记。

用户确认后运行 `confirm-locations`，第二轮才创建逐句精读笔记。所有术语卡写入 confirmed manifest 中的：

```text
<concept_library_root>\概念卡
```

不得再根据文件夹名称猜测“扫盲班”，也不得把路径写死在本 Skill 中。

# 其他规则

- 每个自然段一篇笔记。
- 摘要按一个自然段处理。
- PDF++ 链接必须精确到 selection。
- 中译、精读和术语卡职责分开。
- 笔记默认不在正文重复文件名标题。
